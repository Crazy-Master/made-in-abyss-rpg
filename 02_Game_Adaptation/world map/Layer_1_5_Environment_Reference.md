# Layer 1–5 Environment Reference (Справочник среды слоёв 1–5)

## Кратко

Этот документ — рабочий справочник среды первых пяти слоёв Бездны.

Он нужен для разработки мест, переходов, маршрутов, биомов и будущих игровых сцен.

Документ не заменяет карточки самих слоёв. Он собирает краткую выжимку по тому, как слои выглядят, ощущаются и могут использоваться в игре.

## Принцип использования

Справочник делит сведения на три уровня:

```text
Каноническое ядро — то, что известно из манги, аниме и основных справочных сводок.

Game canon — сведения из Made in Abyss: Binary Star Falling into Darkness. Использовать можно, но осторожно и с пометкой.

Игровая адаптация — наши выводы и предложения для мест, переходов и маршрутов.
````

Главная задача:

```
Сначала понять атмосферу слоя.

Потом выделить биомы и типы мест.

Потом строить маршруты и переходы.
```

## Место и переход

**Место** — игровая зона, где персонажи могут остановиться, осмотреться, собрать образцы, встретить следы, принять решение или столкнуться с событием.

**Переход** — направленное перемещение между местами: спуск, подъём, обход, карниз, корневой путь, проход через трещину, путь по краю чаши, тоннель или переправа.

Рабочая формула:

```
Место даёт сцену.

Переход даёт направление, выбор и цену перемещения.

Маршрут — цепочка мест и переходов.
```

# Layer 1 — Edge of the Abyss (Край Бездны)

## Каноническое ядро

Глубина: 0–1350 м.

Ранг допуска: Red Whistle.

Проклятие при подъёме: лёгкое головокружение и тошнота.

Первый слой расположен прямо под городом [[Orth|Орт]]. Это самый верхний и наиболее понятный участок Бездны. Он считается относительно безопасным по меркам Бездны и используется начинающими Delvers для первых спусков, поиска простых реликвий и учебных заданий.

Ключевые ориентиры:

- [[Gate_to_the_Netherworld|Врата в Преисподнюю]];
- [[Big_Gondola|Большая гондола]];
- [[Burial_Tower|Погребальная башня]];
- [[Praying_Skeletons|Молящиеся скелеты]];
- [[Stone_Ark|Каменный ковчег]].

Среда первого слоя меняется слабо. Здесь есть травянистые площадки, каменные выступы, водопады, трещины, редкая древесная растительность, старые маршруты Delvers и зоны, где уже бывали дети из приютов.

## Визуальный образ

Первый слой должен выглядеть почти знакомо: зелень, камень, водопады, свет сверху, открытые участки, старые тропы, следы работы Delvers.

Но даже здесь Бездна не должна ощущаться обычным лесом. Вертикальность, странные каменные структуры, древние останки и первые реликвии должны постоянно напоминать, что это не поверхность.

## Атмосфера

Тон слоя:

```
учебный, светлый, почти понятный, но уже опасный
```

Здесь игроки должны впервые почувствовать:

- город остался наверху;
- возвращение требует сил;
- рельеф важнее прямой дороги;
- даже простая находка может быть ценной;
- флора и фауна являются частью среды, а не декорацией.

## Возможные биомы для игры

### Старые учебные тропы

Безопасные или условно безопасные маршруты, которыми пользуются Red Whistles. Здесь могут быть следы предыдущих групп, старые метки, остатки верёвок, простая флора и знакомые ориентиры.

### Каменные уступы

Низкие перепады, наклонные плиты, места для первого опыта подъёма и спуска. Хорошо подходят для проверки снаряжения, маршрутного листа и картографии.

### Влажные трещины

Места с капельной водой, мхом, скользким камнем и мелкой фауной. Здесь хорошо работают [[Dewcup_Moss|Каплемох]] и другие индикаторные растения.

### Сухие старые поверхности

Камни, где давно ходили Delvers. Здесь можно использовать лишайники, старые следы, потёртые участки, сухие выступы и слабые намёки на старые маршруты.

### Зоны старых останков

Небольшие участки, связанные с Praying Skeletons или отдельными костными находками. Их лучше использовать осторожно, как лорный и атмосферный элемент, а не как постоянную опасность.

## Рельеф и перемещение

Перемещение относительно простое, но уже не плоское.

Типичные переходы:

- спуск по старой тропе;
- обход каменного уступа;
- короткий подъём по корням или траве;
- проход вдоль влажной стены;
- пересечение открытого склона;
- обход старой осыпи;
- путь к ориентиру.

## Ориентиры

Подходящие ориентиры:

- вид на Орт сверху;
- свет с поверхности;
- Big Gondola;
- Burial Tower;
- Stone Ark;
- старые делверские метки;
- сухие лишайники на старых тропах;
- влажные растения у трещин;
- шум водопада.

## Опасности среды

Опасность первого слоя мягкая, но реальная:

- потеря времени;
- скользкий камень;
- неверная отметка маршрута;
- испорченные образцы;
- мелкая фауна;
- случайный хищник с нижних слоёв;
- первая тошнота и головокружение при подъёме.

## Game canon

В игре первый слой расширяется за счёт дополнительных зон, ресурсов, флоры, фауны и реликвий. Эти сведения можно использовать для наполнения учебных маршрутов, но не стоит считать каждую игровую зону обязательной частью канона.

Полезно для игры:

- дополнительные точки сбора;
- простые ресурсы;
- ранние существа;
- учебные маршруты;
- мелкие реликвии;
- вариативные участки для первых заданий.

## Игровое применение

Layer 1 нужен для первых экспедиций.

Здесь должны работать:

- маршрутный лист;
- простая картография;
- первое снаряжение;
- сбор флоры;
- первый риск фауны;
- первый намёк на Проклятие;
- первая находка реликвии;
- первая ошибка, которую ещё можно пережить.

## Пробелы для homebrew

Можно спокойно добавлять:

- учебные тропы;
- старые делверские стоянки;
- мелкие пещеры;
- влажные карманы;
- безопасные площадки;
- зоны сбора образцов;
- маленькие опасные обходы;
- потерянные записи начинающих Delvers.

# Layer 2 — Forest of Temptation (Лес Искушения)

## Каноническое ядро

Глубина: 1350–2600 м.

Ранг допуска: Blue Whistle.

Проклятие при подъёме: сильная тошнота, головная боль, онемение конечностей.

Второй слой — первый по-настоящему опасный участок Бездны. Среда резко меняется: огромная растительность, лес, влажность, плотная вертикальная структура, опасная фауна и сложная ориентация.

Ключевые ориентиры:

- [[Inverted_Forest|Перевёрнутый лес]];
- [[Heaven_Waterfall|Небесный водопад]];
- [[Giant_Porta_Tree|Гигантское дерево Порта]];
- [[Seeker_Camp|Лагерь Искателей]];
- участки, где ветви деревьев указывают к центру Бездны;
- [[Amagiri|Мох-указчик]].

В слое также упоминается болезнь Tachikiri Fever, против которой опытные Delvers получают защиту.

## Визуальный образ

Второй слой должен выглядеть как лес, который перестал быть просто лесом.

Здесь много крупных растений, тумана, корней, влажных полостей, закрытых проходов, перевёрнутых платформ и странного движения воздуха.

Небо и направление становятся менее очевидными. Игроки должны чаще спрашивать себя:

```
Где верх?

Где центр Бездны?

Куда ведёт этот корень?

Это путь или ловушка?
```

## Атмосфера

Тон слоя:

```
лес живёт по своим правилам, и человек в нём не главный
```

Второй слой должен ощущаться как граница между знакомым и чужим. Здесь уже нельзя просто идти вниз. Нужно читать среду, пользоваться ориентирами и учитывать фауну.

## Возможные биомы для игры

### Верхний лес

Переходная зона после первого слоя. Много крупных растений, плотные кроны, влажная земля, растущая сложность ориентации.

### Туманные корневые карманы

Закрытые участки между корнями, где держится влажный воздух. Хороши для индикаторной флоры, мелкой фауны, скрытых проходов и коротких опасных остановок.

### Вертикальные стволовые маршруты

Переходы по деревьям, корням, лианам и уступам. Здесь хорошо работают [[Redhook_Vine|Красный зацеп]], [[Mistvein_Fern|Туманная жилка]] и другие растения-ориентиры.

### Перевёрнутый лес

Зона сильных восходящих потоков и странной гравитационной/пространственной логики. Вода может течь вверх, платформы кажутся перевёрнутыми, а Force Field слабее обычного.

### Зона Seeker Camp

Более стабильный участок вокруг Giant Porta Tree. Здесь может быть больше следов Delvers, безопасных стоянок, старых маршрутов и дисциплины Ozen.

## Рельеф и перемещение

Перемещение становится сложнее.

Типичные переходы:

- путь по корням;
- спуск вдоль ствола;
- прыжок или переправа между платформами;
- обход через туманный карман;
- проход под крупными листьями;
- подъём по лианам;
- выход к Небесному водопаду.

## Ориентиры

Подходящие ориентиры:

- направление ветвей к центру Бездны;
- [[Amagiri|Амагири]];
- сквозняки;
- туман;
- влажные папоротники;
- светящиеся налёты;
- шум Небесного водопада;
- Giant Porta Tree;
- старые делверские следы вокруг лагеря.

## Опасности среды

Опасности второго слоя:

- потеря ориентации;
- туман;
- влажные корни;
- опасные звери;
- сильная тошнота при подъёме;
- онемение конечностей;
- ложные удобные пути;
- закрытые полости;
- риск остаться без помощи, если Red Whistle спустился слишком глубоко.

## Game canon

Игра расширяет второй слой дополнительными зонами, существами, ресурсами и маршрутами. Эти сведения можно использовать как источник названий, переходов и малых биомов.

Полезно для игры:

- дополнительные точки у Небесного водопада;
- отдельные зоны Inverted Forest;
- игровые маршруты к третьему слою;
- встречающаяся флора и фауна;
- распределение ресурсов по влажным и лесным зонам.

## Игровое применение

Layer 2 нужен, чтобы усилить роль наблюдения.

Здесь игроки должны:

- сверяться с ориентирами;
- читать флору;
- замечать фауну до столкновения;
- понимать, что прямой путь может быть плохим;
- учитывать туман, высоту и влажность;
- бояться не только монстров, но и потери направления.

## Пробелы для homebrew

Можно добавлять:

- корневые мосты;
- закрытые туманные ниши;
- старые тропы с ошибочными метками;
- безопасные деревья-стоянки;
- фауна-кормовые зоны;
- вертикальные переходы;
- малые водопады;
- участки слабого Force Field.

# Layer 3 — Great Fault (Великий разлом)

## Каноническое ядро

Глубина: 2600–7000 м.

Ранг допуска: Moon Whistle.

Проклятие при подъёме: эффекты предыдущих слоёв, а также сильное головокружение, зрительные и слуховые галлюцинации.

Третий слой — огромный вертикальный разлом глубиной около 4000 м. Центральная шахта здесь уже, чем на других слоях. Основное движение идёт не по открытому центру, а через сеть каверн, нор, отверстий и боковых ходов.

Ключевые ориентиры:

- [[Great_Fault|Великий разлом]];
- [[Imprisoned_Pirate_Ship|Заточённый пиратский корабль]];
- боковые каверны;
- норы Neritantan;
- стены с яйцами Kazura Squid ближе к нижней границе;
- выход к четвёртому слою.

## Визуальный образ

Третий слой должен ощущаться не как лес или пещера, а как вертикальная рана в земле.

Основные элементы:

- отвесные стены;
- боковые отверстия;
- узкие уступы;
- сухой ветер;
- скальные карманы;
- тёмные каверны;
- неожиданные открытые провалы;
- древние объекты, застрявшие в стенах.

## Атмосфера

Тон слоя:

```
жизнь не украшает разлом, а цепляется за него
```

Здесь всё связано с маршрутом, высотой и возвращением. Ошибка часто означает не “тебя укусили”, а “ты оказался не там, где думал, и назад идти сложнее”.

## Возможные биомы для игры

### Открытая вертикальная шахта

Опасные стены, ветер, обзор вниз и вверх, летающая фауна. Подходит для сцен страха высоты и выбора пути.

### Боковые каверны

Пещерные карманы, норы, узкие проходы, следы фауны. Здесь могут быть безопасные обходы или ловушки маршрута.

### Ветровые щели

Места, где воздух проходит через скальные полости. Хорошо подходят для [[Hollowstem_Reed|Свистостебля]], сухой флоры, звуковых ориентиров и скрытых проходов.

### Сухие уступы

Места для короткой остановки, сбора аварийной еды, ремонта снаряжения и оценки маршрута.

### Зоны падений

Участки с костями, остатками, корневыми коврами и следами падальщиков. Хорошо подходят для [[Rustroot_Mat|Ржавого ковра]].

### Старые застрявшие объекты

Крупные древние остатки, вроде Imprisoned Pirate Ship, или меньшие homebrew-аналоги: фрагмент конструкции, старая платформа, обломок реликтового механизма.

## Рельеф и перемещение

Перемещение — главная тема слоя.

Типичные переходы:

- карниз вдоль стены;
- спуск по корням;
- проход через норы;
- обход открытой шахты;
- подъём по каменным выступам;
- путь через старый обломок;
- короткая переправа между кавернами;
- спуск к нижнему ярусу.

## Ориентиры

Подходящие ориентиры:

- форма центральной шахты;
- звук ветра;
- направление сквозняка;
- сухие растения;
- тени от нависающих стен;
- следы Neritantan;
- костные участки;
- старые предметы Delvers;
- видимые яйца Kazura Squid внизу.

## Опасности среды

Опасности третьего слоя:

- падение;
- узкие карнизы;
- летающие хищники;
- ложные голоса и визуальные искажения при Проклятии;
- потеря маршрута в кавернах;
- ненадёжные корни;
- ветер;
- усталость;
- нехватка воды;
- возвращение по уже повреждённому пути.

## Game canon

Игра расширяет третий слой большим числом регионов и переходов.

Полезные игровые зоны:

- Great Fault;
- Trapped Pirate Ship;
- Hexagon Pillars;
- Cold Heat Layer;
- Quadruple Pit;
- Glass Layer;
- Rock Slide Hall;
- Freezing Hot Mines;
- Updraft Wasteland;
- Heaven's Waterfall как связь со вторым слоем.

Эти зоны можно использовать как набор возможных биомов и маршрутов, но не обязательно переносить полностью.

## Игровое применение

Layer 3 нужен для маршрутового давления.

Здесь игроки должны:

- планировать путь;
- думать о подъёме заранее;
- понимать цену груза;
- выбирать между открытым опасным путём и закрытой неизвестной каверной;
- сталкиваться с галлюцинациями и неверным восприятием;
- ценить воду, тень и надёжную опору.

## Пробелы для homebrew

Можно добавлять:

- малые каверны;
- корневые мосты;
- ветровые тоннели;
- фрагменты старых объектов;
- обрывы с ложными тропами;
- костные уступы;
- безопасные, но труднодоступные точки отдыха;
- переходы, которые становятся опаснее после повреждения.

# Layer 4 — Goblets of Giants (Кубки Гигантов)

## Каноническое ядро

Глубина: 7000–12000 м.

Ранг допуска: Black Whistle или Moon Whistle в отдельных источниках.

Проклятие при подъёме: сильная боль во всём теле и кровотечение из всех отверстий.

Четвёртый слой — влажные джунгли гигантских растительных чаш. Его главный образ создают [[Flat_Creeper|Чаши Гиганта]] и [[Kenzan_Creeper|Стебли Гиганта]]. Зрелые чаши содержат горячую воду, незрелые — кислоту. Влажность, туман и преломление света через воду создают характерный синий оттенок среды.

Ключевые ориентиры:

- [[Goblets_of_Giants|Кубки Гигантов]];
- [[Garden_of_the_Flowers_of_Fortitude|Сад Цветов Стойкости]];
- [[Nanachi_Hideout|Убежище Наначи]];
- горячие чаши;
- кислотные чаши;
- гигантские растительные ярусы.

## Визуальный образ

Четвёртый слой должен выглядеть красиво и неправильно.

Основные элементы:

- гигантские чаши с водой;
- пар;
- влажный воздух;
- туман;
- синеватый свет;
- цветочные поля;
- огромные живые поверхности;
- скрытая фауна;

## Атмосфера

Тон слоя:

```
красота, которой нельзя доверять
```

Здесь игроки должны видеть, что место великолепно, но почти всё в нём может убить через ошибку: вода, цветы, туман, край чаши, фауна, кислотные участки.

## Возможные биомы для игры

### Зрелые водные чаши

Большие растительные чаши с горячей водой, паром, влажными краями и водной фауной.

### Незрелые кислотные чаши

Опасные участки с кислотной жидкостью, резким запахом, повреждённой органикой и нестабильными краями.

### Паровые низины

Места между чашами, где видимость хуже, а звук искажается влажностью.

### Цветочные поля

Красивые, но опасные поля Eternal Fortunes и похожей флоры. Хорошо подходят для сцены с Amaranthine Deceptor.

### Скрытые безопасные зоны

Удалённые от центральной шахты участки, где Force Field слабее или Проклятие почти не проявляется. Nanachi's Hideout — ключевой пример.

### Живые ярусы

Переходы по стеблям, краям чаш, влажным растительным стенам и корням.

## Рельеф и перемещение

Перемещение нестабильное и живое.

Типичные переходы:

- движение по краю чаши;
- спуск по Kenzan-Creeper;
- обход горячего водоёма;

- путь через цветочное поле;

- обход кислотной чаши;

## Игровое применение

Layer 4 нужен для опасной красоты.

Здесь игроки должны:

- не доверять красивому виду;
- думать о воде как об опасной среде;
- различать горячую воду и кислоту;
- бояться скрытой фауны;
- учитывать влажность, пар и скользкие края;
- понимать, что карта может устареть;
- впервые ощутить, что “место” само является угрозой.

# Layer 5 — Sea of Corpses (Море Останков)

## Каноническое ядро

Глубина: 12000–13000 м.

Ранг допуска: White Whistle или Black Whistle при особых условиях.

Проклятие при подъёме: потеря чувств, спутанность, самоповреждение.

Пятый слой — самый короткий по вертикали, но один из самых широких. Он выглядит как замёрзшая пустошь вокруг огромного моря, дно которого усеяно костями. Это последний слой, с которого ещё физически возможно вернуться на поверхность без потери человечности или смерти.

Ключевые ориентиры:

- [[Ido_Front|Идо Фронт]];
- Altar of the Absolute Boundary;
- море костей;
- кристаллизованные водные опоры;
- водоворот/центральная структура;
- граница перехода к шестому слою.

## Визуальный образ

Пятый слой должен ощущаться холодным, огромным и пустым.

Основные элементы:

- лёд;
- тёмная вода;
- кости;
- замёрзшая пустошь;
- водные кристаллы;
- странные опоры;
- дальняя линия моря;
- искусственная структура Ido Front.

Это уже не “дикая природа”, а граница мира Delvers и почти ритуальная зона последнего возвращения.

## Атмосфера

Тон слоя:

```
холодная граница возврата
```

Здесь мало места для обычного приключенческого тона. Всё должно ощущаться как порог: ниже начинается территория, откуда возвращение уже почти невозможно.

## Возможные биомы для игры

### Замёрзшая пустошь

Холодные открытые участки вокруг моря. Мало укрытий, много ветра, льда, песка, костей и следов старых проходов.

### Берег Моря Останков

Граница воды и суши. Здесь можно находить кости, выброшенные остатки, следы морских существ, старые предметы Delvers.

### Водные кристаллы и опоры

Странные структуры из кристаллизованной воды или “frost”. Могут служить ориентирами, укрытиями, переходами или опасными ледяными зонами.

### Зона Ido Front

Искусственная/ритуальная структура на глубине около 13000 м. Здесь должны ощущаться контроль, технология, культ, лаборатория и власть Bondrewd.

### Граница абсолютного спуска

Места, связанные с переходом к шестому слою. Не просто лифт, а психологическая и лорная граница.

## Рельеф и перемещение

Перемещение широкое и холодное.

Типичные переходы:

- путь по ледяной равнине;
- обход костяного берега;
- движение между водными кристаллами;
- переход к Ido Front;
- спуск к ритуальной структуре;
- путь вдоль моря;
- движение через область с плохой видимостью.

## Ориентиры

Подходящие ориентиры:

- море;
- кости;
- Ido Front;
- водные кристаллы;
- искусственный свет;
- ледяные опоры;
- водоворот;
- линия берега;
- следы Umbra Hands;
- старые маршруты White Whistles.

## Опасности среды

Опасности пятого слоя:

- холод;
- потеря чувств при Проклятии;
- спутанность;
- самоповреждение;
- вода и неизвестная глубина;
- кости под ногами;
- ледяные поверхности;
- древние морские существа;
- невозможность легко вернуться;
- власть Ido Front и Bondrewd.

## Game canon

Игра расширяет пятый слой несколькими зонами.

Полезные игровые зоны:

- Sandy Ice Area;
- Ido Front Area;
- Ido Front Internal;
- Water Crystals;
- Sandstone Region;
- Crystallized Water Supports.

Эти зоны можно использовать как основу для будущей географии пятого слоя.

## Игровое применение

Layer 5 пока не является текущей игровой целью, но справочник нужен заранее.

В будущем этот слой может использоваться для:

- экспедиций высокого уровня;
- контакта с Ido Front;
- сцен последней границы;
- поиска редких реликвий;
- морально тяжёлых решений;
- столкновения с последствиями исследований Bondrewd;
- подготовки к шестому слою.

## Пробелы для homebrew

Можно добавлять:

- ледяные стоянки;
- костяные отмели;
- разбитые старые лагеря;
- малые водные кристаллы;
- зоны плохой чувствительности;
- участки, где Проклятие особенно опасно;
- внешние структуры Ido Front;
- следы неудачных последней экспедиций.

# Общая градация атмосферы

|Слой|Основной образ|Игровой смысл|
|---|---|---|
|Layer 1|зелень, камень, первые тропы|обучение среде|
|Layer 2|лес, туман, корни, вертикальность|ориентация и потеря контроля|
|Layer 3|разлом, ветер, каверны, высота|маршрут и цена ошибки|
|Layer 4|чаши, вода, пар, цветы|красота как опасность|
|Layer 5|лёд, море, кости, Ido Front|граница возврата|

## Общая логика маршрутов

Layer 1:

```
маршруты короткие, учебные, с понятными ориентирами
```

Layer 2:

```
маршруты ветвятся, путаются и требуют чтения среды
```

Layer 3:

```
маршруты становятся вертикальными, однонаправленными и опасными при возврате
```

Layer 4:

```
маршруты проходят через живой изменчивый ландшафт
```

Layer 5:

```
маршруты становятся широкими, холодными и психологически тяжёлыми
```

## Следующее применение документа

После этого справочника можно делать:

1. шаблон карточки места;
2. шаблон карточки перехода;
3. логическую карту Layer 1;
4. первый набор мест для стартовых экспедиций;
5. привязку флоры к местам и переходам;
6. затем — привязку фауны.
