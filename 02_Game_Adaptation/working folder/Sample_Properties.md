# Sample Properties (Характеристики образцов)

## Кратко

Sample Properties — справочный документ с едиными характеристиками для образцов, которые персонажи могут собирать в [[Abyss|Бездне]].

В рабочем русском переводе можно использовать вариант Характеристики образцов.

Этот документ нужен не как строгая анкета, а как общий словарь. В карточках флоры, фауны, реликвий и мест мы используем только те характеристики, которые действительно нужны конкретному образцу.

Главное правило:

Если характеристика важна для игры — указываем её.

Если характеристика ничего не добавляет — не пишем.
## Что считается образцом

Образец — это часть растения, животного, реликтового материала или среды, которую персонажи могут собрать, сохранить и принести наверх.

Примеры образцов:

- лист;
- сок;
- пыльца;
- корень;
- семя;
- оболочка насекомого;
- кость;
- фрагмент шкуры;
- минеральный осколок;
- слизь;
- вода из особого места;
- газ в герметичной колбе;
- фрагмент реликтового материала.

Образец после сбора считается отдельным предметом и занимает место в контейнере для образцов.

## Единица образца (информационный пункт, не является характеристикой)

Одна единица образца — это минимальный объём материала, который можно сдать, изучить или использовать.

Для упрощения:

```
Одна обычная ёмкость для образца вмещает одну полноценную единицу образца.
```

Если образец имеет размер "маленький", в одну ёмкость может поместиться несколько единиц, но это указывается отдельно.

Если образец большой, он требует отдельного специального крепления или емкости, и/или занимать больше места.

## Размерная категория (информационный пункт, не является характеристикой)

Размерная категория описывает, сколько места образец занимает в снаряжении.

Размер "обычный" не указывается в карточке образца. Он считается значением по умолчанию.

| Размер    | Значение                                                                         |
| --------- | -------------------------------------------------------------------------------- |
| маленький | занимает мало места; требует специальный инструмент для сбора.                   |
| обычный   | значение по умолчанию; одна единица занимает один стандартный контейнер          |
| большой   | требует отдельного контейнера, крепления и/или занимает значимое место в рюкзаке |

## Контейнеры образцов (информационный пункт, не является характеристикой)

Лучшим вариантом, для хранения и транспортировки образцов являются специальные контейнеры и емкости. Хотя допускается и использование обычного рюкзака, но в этом случае большинство образцов потеряют ценность и свойства.

Предварительные типы контейнеров:

| Контейнер                     | Для чего нужен                                                                                                                                                                                                                                                                                                         |
| ----------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| рюкзак                        | подходит для любых твердых, прочных образцов (в каждом рюкзаке есть определенное количество внутренних слотов под снаряжение, в которые можно помещать так же специализированные контейнеры или образцы если допускается их хранение без использования контейнеров. Одна единица объекта занимает в рюкзаке один слот) |
| тряпичный контейнер           | подходит для обычных, не являющихся жидкостями или газами (можно прикрепить на внешние слоты рюкзака)                                                                                                                                                                                                                  |
| прочный контейнер             | хрупкие образцы (так же можно использовать и для обычных, не являющихся жидкостями или газами)                                                                                                                                                                                                                         |
| стеклянная колба с пробкой    | жидкие, летучие, пахучие, газообразные или токсичные образцы образцы (хрупкая, но не подвержена воздействию кислот)                                                                                                                                                                                                    |
| металлическая колба с пробкой | жидкие, летучие, пахучие, газообразные или токсичные образцы образцы (ударопрочная, но подвержена воздействию кислот)                                                                                                                                                                                                  |
| масляная капсула              | любые образцы, которые портятся от воздуха или влаги                                                                                                                                                                                                                                                                   |

Контейнеры будут подробнее расписаны в карточках снаряжения.

## Базовые характеристики

### Сбор (обязательная характеристика)

Сбор описывает, как получить образец.

Примеры:

- срезать;
- сорвать;
- соскоблить;
- выжать;
- собрать с поверхности;
- отделить от мёртвого организма;
- взять отпавший фрагмент;
- поймать в колбу;
- аккуратно выкопать.

### Инструмент (обязательная характеристика, если нужен инструмент)

Инструмент описывает, чем удобнее или безопаснее собирать образец.

Единые значения:

| Значение                                | Когда использовать                                           |
| --------------------------------------- | ------------------------------------------------------------ |
| что-то режущее (Нож, заточенный камень) | когда нужно что-то срезать (перерезать) или что-то соскрести |
| пинцет с моноклем                       | мелкие, хрупкие или опасные фрагменты                        |
| кирка                                   | работа с твердыми объектами, а так же для раскопок           |
| колба                                   | жидкость, газ, летучие вещества                              |
| перчатки                                | контактные или раздражающие образцы                          |

Если инструмент не нужен, характеристика не указывается.

### Хранение (обязательная характеристика)

Хранение описывает, где и как держать образец после сбора.

Единые значения:

| Значение                                    | Когда использовать                                                                          | Пример контейнеров                                              |
| ------------------------------------------- | ------------------------------------------------------------------------------------------- | --------------------------------------------------------------- |
| сухое место                                 | сухие и неприхотливые образцы                                                               | рюкзак, тряпичный контейнер, прочный контейнер                  |
| место, защищенное от физических воздействий | хрупкие твёрдые образцы                                                                     | прочный контейнер                                               |
| стеклянное герметичное место                | жидкие, летучие, пахучие, газообразные, кислотные или токсичные образцы образцы             | стеклянная колба с пробкой                                      |
| любая герметичная колба                     | жидкие, летучие, пахучие, газообразные или токсичные образцы образцы не являющиеся кислотой | стеклянная колба с пробкой, металлическая колба с пробкой       |
| отдельно от других образцов                 | если образец пахнет, пачкает, ядовит или портит соседние находки                            | в любом контейнере, желательно герметичном или снаруже рюкзака. |
| темное место                                | если образец портится от света                                                              | любой закрытый контейнер, кроме стеклянной колбы                |
| влажное место                               | если образец быстро высыхает                                                                | любой влажный контейнер                                         |

### Срок годности (не обязательная характеристика)

Срок годности описывает, через сколько образец портится или теряет ценность. Если не портится при обычном хранении то эту характеристику не указываем.

Примеры значений:

- 1 час;
- 6 часов;
- 24 часа;
- 3 дня;
- до высыхания;
- пока герметично закрыт;

### Состояние (обязательная характеристика, если какого-то особого состояния нету, то по умолчанию ставится "неповреждённый")

Состояние описывает, в каком виде образец считается пригодным . Он может попадаться как в одном, так и в другом состоянии. При использовании данной характеристики в ней следует указывать текущее значение состояния, а рядом с ним в  скобках состояние в котором образец имеет свойства.

Единые значения:

| Значение                         | Смысл                                                              |
| -------------------------------- | ------------------------------------------------------------------ |
| свежий (старый)                  | свойства сохраняются первый час                                    |
| сухой (мокрый)                   | для получения свойств необходимо высушить образец                  |
| мокрый (сухой)                   | для получения свойств необходимо намочить образец                  |
| неповреждённый<br>(повреждённый) | свойства проявляются только у неповрежденного образца              |
| чистый (грязный)                 | свойства проявляются только у очищенного образца                   |
| зрелый (несозревшый)             | свойства проявляются только у собранного в подходящей стадии роста |
| измельчённый (целый)             | приобретает свойства после измельчения                             |


## Практические характеристики 

### Польза (не обязательная характеристика)

Польза описывает основное применение образца.

Значения:

- пищевой компонент;
- лекарственный компонент (после указания этой характеристики в скобках нужно указать значение, а именно положительный эффект, руководствуясь руководствуемся [[Conditions and Status Effects|Состояния и эффекты]] );
- практическое применение (использовать в качестве клея, веревки, ножа и т.д.);
- приманивает фауну (их будем добавлять, когда будет готова фауна);
- отпугивает фауну (их будем добавлять, когда будет готова фауна);

### Риск (не обязательная характеристика)

Риск описывает негативный эффект или проблему, связанную с образцом.

Значения:

- вызывает негативный эффект (токсичный при контакте, токсичные испарения) (после указания этой характеристики в скобках нужно указать значение, а именно отрицательный эффект, руководствуясь руководствуемся [[Conditions and Status Effects|Состояния и эффекты]] )
- при хранение рядом с пищей портит ее;
- пачкает другие образцы;
- перебивает запахи;
- становится жидким, через какое то время;
- кислотный, разъедает при контакте.

### Риск появляющийся при воздействии чего либо на образец (не обязательная характеристика)

Эта характеристика описывает, те воздействия, которые портят образец или делают его опасным. В тексте с образцом, где используется эта характеристика нужно описать какой риск будет приобретать образец, можно руководствоваться значениями из пункта "Риск".

Значения:

- вода;
- нагревание;
- открытый воздух;
- прямой солнечный свет (свет центра бездны);
- кровь;
- металлический контейнер;
- смешивание с кислотными веществами.


### Крафт, Обработка, Совместимость (не обязательная характеристика, добавим позже)

Эти характеристики мы будем добавлять позже, когда у нас будет готова база по флоре, фауне и реликвиям.

### Ценность (обязательная характеристика)

Ценность описывает примерную ценность образца для исследователей, приюта, Гильдии или торговцев.

Единые значения:

| Значение          | Смысл                                        |
| ----------------- | -------------------------------------------- |
| учебная           | подходит для отчёта или обучения             |
| низкая            | почти не имеет цены, но полезна как сведения |
| средняя           | может заинтересовать оценщика или лекаря     |
| высокая           | редкий или полезный образец                  |
| исследовательская | важен не ценой, а сведениями                 |
| опасная           | ценен, но требует осторожного хранения       |
| запрещённая       | нельзя хранить или продавать без допуска     |

## Физические характеристики

### Запах (не обязательная характеристика)

Запах описывает выраженность и игровое значение запаха. Описывается кратко, но красочно, не более 6-10 слов.


### Вкус (не обязательная характеристика)

Вкус указывается только если образец применим в еде или медицине. Описывается кратко, но красочно, не более 6-10 слов.

### Летучесть (не обязательная характеристика)

Летучесть описывает, насколько быстро образец испаряется.

Единые значения:

| Значение                                                    | Смысл                                          |
| ----------------------------------------------------------- | ---------------------------------------------- |
| низкая (по умолчанию, в характеристике образца не ставится) | не испаряется или испарения не значительны     |
| высокая                                                     | быстро испаряется или распространяет запах     |
| мгновенная                                                  | свойства теряются почти сразу после вскрытия   |
| только при повреждении                                      | активируется после надреза, разлома или сжатия |
| только при нагреве                                          | становится летучим при высокой температуре     |

### Агрегатное состояние (обязательная характеристика)

Агрегатное состояние описывает физическую форму образца.

Твёрдое состояние считается значением по умолчанию и не указывается, если не важно.

Единые значения:

| Значение     | Смысл                                  |
| ------------ | -------------------------------------- |
| твёрдое      | значение по умолчанию                  |
| жидкое       | требует колбу или герметичное хранение |
| газообразное | требует герметичную колбу              |

### Размерная категория (обязательная характеристика)

Размерная категория описывает место, которое образец занимает в снаряжении.

Обычный размер считается значением по умолчанию и не указывается.

Единые значения:

| Значение  | Смысл                                                                            |
| --------- | -------------------------------------------------------------------------------- |
| маленький | занимает мало места; требует специальный инструмент для сбора.                   |
| обычный   | значение по умолчанию; одна единица занимает один стандартный контейнер          |
| большой   | требует отдельного контейнера, крепления и/или занимает значимое место в рюкзаке |

## Формат записи образца в карточке

Рекомендуемый формат (При необходимости, можно и нужно использовать соответствующие не обязательные характеристики. Этот рекомендуемый формат, лишь пример возможного оформления, не обязующий повторять его один в один):

```
- **Название образца** — краткое описание. *Сбор:* ... *Инструмент:* ... *Хранение:* ... *Срок годности:* ... *Польза:* ... *Риск:* ... *Ценность:* ... *Агрегатное состояние:* ... *Размерная категория:*
```

Пример:

```
- **Свежий лист сайноны** — мягкий лист с резким травяным запахом. *Сбор:* аккуратно срезать, не мять. *Инструмент:* что-то режущее. *Хранение:* сухое место. *Срок годности:* 24 часа. *Польза:* слабая пищевая ценность. *Ценность:* учебная. *Агрегатное состояние:* твёрдое. *Размерная категория:* обычный.
```

Не нужно заполнять все характеристики, если логически это не требуется. Образец должен иметь полное описание всех характеристик, которые относятся к нему.
