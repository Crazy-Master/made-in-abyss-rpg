# Interaction Triggers (Триггеры взаимодействия)

## Кратко

Interaction Triggers — справочный документ с типовыми триггерами, которые могут запускать свойства флоры, фауны, реликвий, мест и переходов.

В рабочем русском переводе можно использовать вариант Триггеры взаимодействия.

Триггер — это условие, при котором объект начинает действовать, менять состояние или создавать последствия.

Примеры:

- растение выделяет газ при повреждении;
- реликвия активируется при попадании крови;
- фауна реагирует на шум;
- переход становится опаснее ночью;
- образец портится от воды;
- существо подходит на запах пота.

Документ нужен как общий словарь, а не как жёсткий список обязательных правил.

## Главное правило

Триггеры нужно использовать аккуратно.

Хороший триггер:

- понятен мастеру;
- логичен для объекта;
- может быть замечен или предсказан игроками;
- создаёт выбор;
- не выглядит как произвольное наказание.

Плохой триггер:

- срабатывает без признаков;
- не связан с природой объекта;
- не даёт игрокам шанса понять риск;
- используется только чтобы усложнить сцену.

## Масса и нагрузка

### Крупный предмет

Крупный предмет — персонаж несёт хотя бы один большой предмет, который заметно увеличивает нагрузку или мешает двигаться.

Использование:

- растение не выдерживает нагрузку;
- корневая полость проваливается;
- узкий переход становится сложнее;
- подъём требует больше времени;
- снаряжение цепляется за ветви или камни.

Пример:

Если персонаж несёт большой предмет, поверхность растения начинает рваться при попытке спуска.

### Группа из нескольких человек

Группа из нескольких человек — несколько персонажей одновременно находятся рядом, наступают на одну поверхность или входят в зону действия объекта.

Использование:

- растение реагирует на суммарную нагрузку;
- фауна замечает группу по шуму;
- узкий уступ становится опасным;
- газ быстрее заполняет небольшое пространство;
- падающий камень угрожает сразу нескольким.

Пример:

```
Если на корневую площадку одновременно встают два или больше персонажа, она начинает проседать.
```

## Воздействие на объект

### Прикосновение

Прикосновение — лёгкий контакт с объектом.

Использование:

- листья темнеют;
- поверхность становится липкой;
- растение сворачивается;
- реликвия слегка нагревается;
- фауна реагирует на нарушение границы.

Пример:

```
При прикосновении растение оставляет на коже раздражающий сок.
```

### Повреждение

Повреждение — надрез, прокол, удар, разрыв или грубое воздействие.

Использование:

- растение выделяет газ;
- сок становится токсичным;
- объект издаёт звук;
- реликвия активируется;
- фауна воспринимает действие как угрозу.

Пример:

```
При повреждении стебля растение выпускает сладкий запах, привлекающий мелкую фауну.
```

### Сбор образца

Сбор образца — попытка извлечь часть объекта.

Использование:

- нужно выбрать правильный инструмент;
- можно испортить образец;
- растение может отреагировать защитно;
- сбор оставляет след;
- объект меняет состояние после сбора.

Пример:

```
Если собрать цветок без перчаток, пыльца вызывает слезотечение.
```

### Нагрев

Нагрев — воздействие высокой температуры, огня, горячего металла или пара.

Использование:

- сок испаряется;
- образец теряет свойства;
- газ становится токсичным;
- реликвия активируется;
- фауна реагирует на дым.

Пример:

```
При нагреве смола становится летучей и вызывает кашель в закрытом пространстве.
```

### Охлаждение

Охлаждение — воздействие холода, холодной воды, ночного воздуха или специальных контейнеров.

Использование:

- образец дольше хранится;
- растение застывает;
- сок густеет;
- реликвия временно гаснет;
- фауна теряет интерес.

Пример:

```
Охлаждённый сок сохраняет свойства до трёх дней, но при комнатной температуре портится за сутки.
```

### Попадание воды

Попадание воды — дождь, ручей, мокрая ткань или другая жидкость.

Использование:

- порошок портится;
- растение раскрывается;
- поверхность становится скользкой;
- образец начинает гнить;
- реликвия меняет поведение.

Пример:

```
После попадания воды лист становится скользким и больше не подходит как образец.
```

### Попадание крови

Попадание крови — кровь человека, животного или существа Бездны попадает на объект.

Использование:

- растение привлекает падальщиков;
- реликвия реагирует;
- фауна выходит на запах;
- образец становится загрязнённым;
- объект меняет цвет.

Пример:

```
Кровь на корнях резко усиливает запах растения и может привлечь хищников.
```

## Время

### День

День — обычное светлое время.

Использование:

- растение раскрыто;
- фауна менее активна;
- видимость лучше;
- почтовый шар легче запустить;
- опасности проще заметить.

Пример:

```
Днём цветок раскрыт, и пыльцу можно собрать без повреждения лепестков.
```

### Ночь

Ночь — тёмное время, когда видимость хуже, а часть фауны активнее.

Использование:

- растение закрывается;
- свет привлекает внимание;
- фауна выходит на охоту;
- ориентиры хуже видны;
- подъём становится опаснее.

Пример:

```
Ночью растение закрывает листья, и безопасный сбор становится невозможен без повреждения.
```

### Длительное пребывание рядом

Длительное пребывание рядом — персонажи остаются в зоне действия объекта достаточно долго, чтобы накопился эффект.

Использование:

- газ вызывает сонливость;
- запах привлекает фауну;
- влажность портит снаряжение;
- реликвия постепенно нагревается;
- состояние персонажей ухудшается.

Пример:

```
После нескольких минут рядом с повреждённым растением персонажи начинают ощущать сонливость.
```

### Сезон цветения

Сезон цветения — особое состояние растения или зоны, которое может быть связано с событием.

Использование:

- растение становится опаснее;
- появляются редкие образцы;
- фауна собирается рядом;
- запах усиливается;
- маршрут меняется.

Пример:

```
Во время цветения растение привлекает насекомых и становится ценным источником пыльцы.
```

## Пространство

Эти триггеры чаще используются для фауны, реликвий, мест и переходов, но могут применяться и к флоре.

### Уровень Бездны

Уровень Бездны — глубина, слой или вертикальная зона, где находится объект.

Использование:

- реликвия меняет поведение;
- фауна становится агрессивнее;
- растение встречается в другом подвиде;
- Проклятие усиливается;
- образец становится ценнее.

Пример:

```
На втором слое тот же вид растения содержит более концентрированный сок.
```

### Наличие ветра

Наличие ветра — открытая зона, сквозняк или сильное движение воздуха.

Использование:

- газ рассеивается;
- запах распространяется дальше;
- почтовый шар легче или труднее запустить;
- пыльца разносится по зоне;
- фауна чувствует запах издалека.

Пример:

```
При ветре споры не задерживаются у растения, но могут попасть на соседний уступ.
```

### Высокая влажность

Высокая влажность — туман, водопад, мокрые стены, дождь или влажная полка.

Использование:

- растение активнее;
- поверхность становится скользкой;
- образцы быстрее гниют;
- фауна оставляет больше следов;
- снаряжение отсыревает.

Пример:

```
Во влажной зоне листья растения становятся липкими и начинают цепляться за ткань.
```

## Поведение игроков

### Шум

Шум — громкий разговор, падение камней, крик, бой, удар по металлу, кашель или резкий звук.

Использование:

- фауна замечает группу;
- падальщики отступают или атакуют;
- хищник меняет маршрут;
- камни осыпаются;
- наставник позже может спросить, почему группа привлекла внимание.

Пример:

```
Шум у тела Delver может привлечь падальщиков, которые считают это место своей добычей.
```

### Использование света

Использование света — фонарь, факел, яркая реликвия или отражение от блестящего предмета.

Использование:

- фауна замечает группу;
- свет отпугивает мелких существ;
- блестящие предметы привлекают птиц;
- становится проще найти маршрут;
- некоторые растения закрываются или раскрываются.

Пример:

```
Яркий свет отпугивает мелких падальщиков, но может привлечь внимание летающих существ на открытом участке.
```

## Как записывать триггеры в карточках

Триггеры лучше писать в разделе свойств объекта обычным текстом.

Пример:

```
При повреждении листьев растение выделяет сонливый запах. В открытом месте он быстро рассеивается, но в закрытой нише длительное пребывание рядом может вызвать Сонливость.
```

Для краткой записи можно использовать курсив внутри описания образца:

```
- **Сонная пыльца** — мелкий жёлтый порошок. *Триггер:* повреждение цветка. *Риск:* сонливость при вдыхании. *Хранение:* герметичная колба.
```
