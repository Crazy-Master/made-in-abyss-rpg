# First Training Descent Matrix (Матрица первого учебного спуска)

## Кратко

Этот документ описывает первый учебный спуск воспитанников Белчеро на первый слой Бездны.

Это не полноценные карточки мест и переходов, а проектная матрица.

Цель документа:

```text
спроектировать первый игровой участок;
понять, какие места нужны;
понять, какие переходы их связывают;
заложить сбор образцов, отправку шара, отметку успешного спуска, странную находку и обратный путь;
сделать маршрут не линейным, а сеточным;
подготовить основу для будущих карточек L и T.
````

## Используемые зоны

Первый учебный спуск использует только малую часть первого слоя.

Основные зоны:

```text
Z001 Upper Rim — верхний край, официальный входной пояс.
Z002 Training Slopes — учебные склоны, основная зона первого спуска.
Z004 Wet Crack Line — влажные трещины, боковой учебный фрагмент.
```

Подзоны:

```text
Z001-B First Marked Stones — первые метки и входная учебная линия.
Z002-A Old Training Slope — старый учебный склон.
Z002-B First Sample Shelf — первая полка образцов.
Z002-C Marked Return Line — линия возврата по меткам.
Z004-A Dew Pockets — влажные моховые карманы.
Z004-B Slippery Side Cracks — скользкие боковые трещины.
```

## Игровая задача спуска

Официальная задача для игроков:

```text
спуститься на первый слой;
отметить успешный спуск;
собрать несколько простых образцов;
отправить шар с отчётом;
вернуться в течение 24 часов.
```

Скрытая игровая задача:

```text
показать, что Бездна даже на первом слое не является обычной прогулкой;
дать игрокам почувствовать рельеф, ориентирование, образцы, переходы, скрытые детали, сеточность местности и первое проявление Проклятия.
```

## Масштаб

Ориентировочный масштаб участка:

```text
12 мест;
18 переходов;
1 основной учебный путь;
4 боковых ответвления;
2 скрытых перехода;
2 сеточные петли;
3 точки сбора образцов;
1 учебная зона отправки шара;
1 точка отметки успешного спуска;
1 важная необычная находка;
1 опасный, но соблазнительный срез;
1 мягкое проявление Проклятия на обратном пути.
```

## Принцип сеточности

Основной путь должен быть понятным и отмеченным. Его читают по старым меткам, верёвкам, зарубкам, учебным знакам и естественным ориентирам.

Боковые пути не должны ломать учебную структуру, но должны показывать, что первый слой шире одной тропы.

Игроки должны иметь возможность:

```text
обойти часть маршрута;
ошибиться с похожими полками;
найти скрытый влажный обход;
срезать путь рискованно;
вернуться другим путём;
увидеть, что карта может быть неполной.
```

# Краткая логика графа

Основная учебная линия:

```text
L001 → T001 → L002 → T002 → L003 → T003 → L004 → T004 → L005 → T005 → L006
```

Основная возвратная линия:

```text
L006 → T006 → L007 → T007 → L004 → T003/T002/T001 → L001
```

Сеточные боковые линии:

```text
L002 → T015 → L011 → T016 → L004

L003 → T008 → L008 → T011 → L004

L004 → T009 → L009 → T012 → T004

L005 → T010 → T017 → L012 → T018 → L007
```

Опасный соблазн:

```text
L005 → T013 → L007
```

Длинный запасной возврат:

```text
L007 → T014 → L002
```

## Учебная логика отправки шара

L005 должна быть не просто точкой запуска, а маленькой учебной площадкой.

В ней должно быть несколько вариантов:

```text
открытая площадка — шар легко уходит вверх, но место заметнее;
защищённая ниша — безопаснее от ветра, но шар может задеть верхнюю трещину;
сухой каменный выступ — хороший компромисс, если игроки его заметят.
```

Ошибочный запуск не должен сразу ломать задание, но должен создавать последствия:

```text
шар застревает ненадолго;
нужно сменить площадку;
теряется время;
шум может привлечь мелкую фауну;
отчёт уходит позже, чем хотелось.
```

## Проверка графа

Минимальная структура выполнена:

```text
есть стартовое место;
есть основной путь вниз;
есть точка отметки спуска;
есть точка отправки шара;
есть обратный путь;
есть боковые ответвления;
есть скрытые переходы;
есть рискованный необязательный срез;
есть места сбора образцов;
есть место для Star Compass;
есть мягкое проявление Проклятия на возврате;
есть сеточность до глубины около 230 м;
есть возможность заблудиться без превращения маршрута в лабиринт.
```

## Что нужно проверить перед карточками

Перед созданием полноценных карточек нужно решить:

```text
1. Не слишком ли плотная сетка для первого учебного спуска.
2. Нужно ли объединить T010 и T017 в одну карточку перехода.
3. Достаточно ли явно основной маршрут читается по меткам.
4. Насколько заметным должен быть риск T013 до того, как игроки его выберут.
5. Нужно ли добавить отдельную механику для запуска шара или описать её только в L005.
```

## Следующий шаг

После утверждения матрицы можно создавать карточки в таком порядке:

```text
1. L001 First Marked Stones
2. T001 Marked Stone Descent
3. L002 Shallow Training Shelf
4. T002 Low Slope Traverse
5. L003 Old Training Ledge
6. T003 Old Rope Drop
7. L004 Lower Lesson Shelf
```

Лучше начать с первых 3–4 мест и 3–4 переходов, чтобы проверить формат на практике.
