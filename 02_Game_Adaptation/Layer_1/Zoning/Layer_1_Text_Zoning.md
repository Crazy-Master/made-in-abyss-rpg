# Layer 1 Text Zoning (Текстовое зонирование первого слоя)

## Кратко

Этот документ описывает рабочее зонирование первого слоя Бездны.

Он нужен для разработки подзон, матриц мест, матриц переходов и маршрутов-маяков.

Это не карта для игроков и не полный список всех мест первого слоя. Это проектная структура, на основе которой позже создаются карточки мест и переходов.

Главная логика:

```
Layer 1 = крупные зоны.

Каждая зона = несколько подзон.

Каждая подзона = набор мест и переходов.
```

## Масштаб

Первый слой огромен. Диаметр Бездны у поверхности около 1 км, а ниже пространство постепенно расширяется.

Поэтому первый слой нельзя проектировать как одну дорогу вниз. Даже “учебная” область первого слоя должна быть лишь малой частью общей структуры.

В этом документе зонирование делится на:

```
входы;
крупные зоны;
подзоны;
маршруты-маяки;
заметки для будущих мест и переходов.
```

## Каноническое ядро Layer 1

Глубина:

```
0–1350 м
```

Роль:

```
верхний, наиболее доступный и учебный слой Бездны
```

Проклятие при подъёме:

```
лёгкое головокружение и тошнота
```

Ключевые ориентиры:

```
Gate to the Netherworld;
Big Gondola;
Burial Tower;
Praying Skeletons;
Stone Ark.
```

Общий тон:

```
светлый, учебный, почти понятный, но уже опасный
```

Layer 1 должен учить игроков читать среду: рельеф, следы, старые метки, влажность, флору, фауну, состояние переходов и цену возврата.

# Входы в первый слой

## E001 Gate to the Netherworld

**Врата в Преисподнюю.**

survey_level: `mapped`

Роль: официальный вход в Бездну из Орта.

Это главный официальный вход, через который Delvers начинают спуск. Он связан с контролем, разрешениями, гильдией и обычной структурой профессионального исследования.

Через этот вход логично начинать первый учебный спуск воспитанников Белчеро.

Ведёт к зонам:

```
Z001 Upper Rim;
Z002 Training Slopes;
частично к официальным учебным маршрутам.
```

## E002 Big Gondola / Grand Pier Line

**Большая гондола / линия Большого причала.**

survey_level: `mapped`

Роль: инфраструктурный спуск вглубь первого слоя.

Гондольная линия связывает Орт и верхнюю инфраструктуру первого слоя с нижними ориентирами, примерно к области Stone Ark.

Для первых учебных спусков это не основной путь, но важный ориентир мира. Она показывает, что первый слой частично освоен и встроен в работу Орта.

Ведёт к зонам:

```
Z003 Gondola Infrastructure Belt;
Z007 Stone Ark Zone.
```

## E003 Wharf Side Access

**Боковой доступ со стороны пристани.**

survey_level: `rough / hidden`

Роль: неофициальный, обходной или плохо контролируемый доступ.

Этот вход нужен не для первого учебного спуска, а для будущих сюжетов: побег, нелегальный спуск, чужие Delvers, контрабанда, обход контроля.

Ведёт к зонам:

```
Z008 Wild Side Slopes;
Z009 Unmarked Outer Ridges.
```

# Маршруты-маяки

Маршруты-маяки — это не правильные пути для игроков.

Это проектные линии, которые помогают понимать, где какие зоны должны находиться и какую функцию они выполняют.

## B001 official_training

Официальные учебные пути Red Whistles.

Линия:

```
Gate to the Netherworld → Upper Rim → Training Slopes → First Descent Practice Area
```

Функция:

```
первое снаряжение;
картография;
сбор образцов;
отметка успешного спуска;
возврат до истечения срока.
```

Это главный маяк для первого учебного спуска.

## B002 gondola_beacon

Инфраструктурная линия гондолы.

Линия:

```
Orth / Grand Pier → Big Gondola → Stone Ark Zone
```

Функция:

```
ориентир освоенности слоя;
быстрый профессиональный спуск;
связь верхнего Орта с нижними зонами первого слоя.
```

## B003 ruin_beacon

Древние структуры и лорные ориентиры.

Линия:

```
Training Slopes → Burial Tower Zone → Praying Skeletons-related areas
```

Функция:

```
древность Бездны;
намёк на циклы;
мрачный контраст с учебным тоном первого слоя.
```

## B004 wet_beacon

Влажные трещины, водопады и моховые участки.

Линия:

```
Upper Rim → Wet Cracks → Waterfall Slopes → Stone Ark side areas
```

Функция:

```
вода;
мох;
скользкий камень;
мелкая фауна;
боковые обходы;
ошибки картографии.
```

## B005 wild_beacon

Боковые и плохо изученные участки.

Линия:

```
Wharf Side Access → Outer Ridges → Wild Side Slopes
```

Функция:

```
меньше меток;
меньше контроля;
выше риск;
больше скрытых переходов;
возможные следы чужих Delvers.
```

## B006 lower_edge_beacon

Нижняя граница первого слоя.

Линия:

```
Stone Ark Zone → Lower Edge → approach to Layer 2
```

Функция:

```
переход к более серьёзной игре;  
усиление опасности;  
первое ощущение, что учебная зона закончилась.
```

# Крупные зоны Layer 1

## Z001 Upper Rim

**Верхний край.**

Высота:

```
0–100 м
```

survey_level: `mapped`

Основные маяки:

```
official_training;
gondola_beacon.
```

Ключевая функция:

```
официальный входной пояс и первые безопасные участки.
```

Визуальный образ:

```
верхний свет, вид на Орт, старые метки, первые каменные уступы, следы постоянного движения Delvers.
```

Типичные места:

```
официальная входная площадка;
первые каменные полки;
малые учебные уступы;
участки со старыми метками;
простые точки осмотра.
```

Типичные переходы:

```
короткие спуски;
старые тропы;
низкие каменные ступени;
простые обходы;
первые верёвочные участки.
```

Риски:

```
минимальные по меркам Бездны;
неверная самоуверенность;
мелкие срывы;
ошибки с первыми образцами.
```

Заметки:

```
Это не зона для серьёзной угрозы. Она нужна, чтобы показать, что Бездна начинается сразу под городом, но пока ещё контролируется.
```

## Z002 Training Slopes

**Учебные склоны.**

Высота:

```
100–250 м
```

survey_level: `patrolled`

Основные маяки:

```
official_training;
wet_beacon.
```

Ключевая функция:

```
главная зона для первого учебного спуска.
```

Визуальный образ:

```
каменные склоны, небольшие уступы, старая трава, влажные трещины, простые спуски, следы детских и взрослых Delvers.
```

Типичные места:

```
старый учебный уступ;
площадка для отметки успешного спуска;
нижняя полка для сбора образцов;
малый лагерь;
влажная стенка;
открытая обзорная площадка.
```

Типичные переходы:

```
старый верёвочный спуск;
короткий каменный траверс;
влажная трещина;
пологий склон;
узкий обход между камнями.
```

Риски:

```
скользкий камень;
ненадёжная старая верёвка;
испорченные образцы;
мелкая фауна;
ошибка при возвращении;
первое проявление Проклятия.
```

Заметки:

```
Это первая зона, которую нужно детализировать местами и переходами. Для первого учебного спуска достаточно разработать часть этой зоны и 1–2 соседние подзоны.
```

## Z003 Gondola Infrastructure Belt

**Гондольный инфраструктурный пояс.**

```

survey_level: `mapped`

Основные маяки:

gondola_beacon.
```

Ключевая функция:

```
показать освоенность первого слоя и связь Орта с Бездной.
```

Визуальный образ:

```
крупные тросы, деревянные или металлические конструкции, старые платформы, шум механизмов, следы профессионального обслуживания.
```

Типичные места:

```
верхняя гондольная площадка;

```

```
меньше природной опасности;

опасность падения с технических конструкций;
закрытые зоны для новичков.

Заметки:

```
Для первого учебного спуска гондола скорее дальний ориентир или элемент мира, а не основной маршрут.

# Рекомендованная зона для первого учебного спуска

Для первого учебного спуска лучше использовать:

```
Z001 Upper Rim;
Z002 Training Slopes;
малую часть Z004 Wet Crack Line.
```

Не использовать напрямую:

```
Z005 Burial Tower Zone;
Z007 Stone Ark Zone;
Z008 Wild Side Slopes;
Z009 Lower Edge.
```

Почему:

```
первый спуск должен быть учебным;
игроки должны освоить карту, образцы, переходы и возврат;
риск должен быть настоящим, но не перегруженным;
глубокие ориентиры лучше оставить как будущие цели.
```

## Черновая структура первого учебного спуска

Ориентировочный масштаб:

```
8–12 мест;
10–16 переходов;
1 основной учебный путь;
2–4 боковых ответвления;
1–2 скрытых перехода;
2–3 точки сбора образцов;
1 важная необычная находка;
1 точка отправки шара;
1 точка для отметки успешного спуска;
1 обратный путь с первым проявлением Проклятия.
```

Зоны:

```
старт: Z001-A Gate Mouth или Z001-B First Marked Stones;
основная игра: Z002-A Old Training Slope;
сбор образцов: Z002-B First Sample Shelf;
возврат: Z002-C Marked Return Line;
боковой путь: Z004-A Dew Pockets или Z004-B Slippery Side Cracks.
```

## Возможные функции стартовых мест

Для будущей матрицы мест:

```
первый обзорный уступ;
старая учебная площадка;
полка сбора образцов;
влажная стенка с мхом;
малый привальный карман;
место отметки успешного спуска;
точка отправки шара;
боковая расщелина;
место странной находки;
место первого следа фауны.
```

## Возможные функции стартовых переходов

Для будущей матрицы переходов:

```

старый верёвочный спуск;

небольшой рискованный срез.
```

# Принцип расширения Layer 1

После первого учебного спуска слой расширяется не сразу во все стороны, а по соседним зонам.

Порядок расширения:

```
1. Z002 Training Slopes.
2. Z004 Wet Crack Line.
3. Z006 Middle Descent Belt.
4. Z005 Burial Tower Zone.
5. Z007 Stone Ark Zone.
6. Z008 Wild Side Slopes.
7. Z009 Lower Edge.
```

Так игроки постепенно переходят от учебной среды к более самостоятельному исследованию.

## Главное правило

```
Каждая новая зона должна отличаться от предыдущей не только названием, но и игровой функцией.
```

Если зона не даёт нового типа решений, рисков, ориентиров или атмосферы, её лучше объединить с соседней зоной или оставить как подзону
