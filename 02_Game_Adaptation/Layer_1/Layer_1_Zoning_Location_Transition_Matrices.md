# Layer 1 Zoning, Location and Transition Matrices

## Кратко

Этот документ нужен для проектирования первого слоя перед созданием полноценных карточек.

Здесь фиксируются:

```
крупные зоны;
места;
переходы;
проектные маяки;
уровень изученности;
роли будущих сцен.
```

Матрицы не являются финальными карточками. Они нужны, чтобы быстро увидеть структуру слоя и не создавать места хаотично.

## Рабочая логика

```
Слой → зоны → места → переходы → маршруты Delvers → полноценные карточки
```

Маршруты Delvers здесь рассматриваются не как единственно правильные пути, а как проектные маяки. Они помогают понять, где должны быть основные, боковые, скрытые, опасные и учебные участки.

## Survey level

`survey_level` показывает, насколько зона, место или переход изучены Delvers.

Возможные значения:

```
mapped — хорошо нанесено на карты и часто используется
patrolled — регулярно посещается или проверяется
known — известно, но не контролируется
rough — описано плохо
wild — почти не изучено
forbidden — опасно, закрыто или формально запрещено
```

Общее правило:

```
Чем дальше участок от основных троп, тем ниже survey_level и тем выше риск.
```

## Availability

`availability` показывает роль элемента в будущей игровой структуре.

Возможные значения:

```
main_route — основной элемент маршрута
side_route — боковая зона или боковой путь
hidden — скрытый элемент
optional — необязательный элемент
dangerous — опасный элемент
special — сюжетно или событийно важный элемент
```

# Как использовать матрицы

Сначала заполняется матрица зон.

Потом для каждой зоны создаются черновые места.

Потом между местами добавляются переходы.

После этого проверяется граф:

```
есть ли основной маяк;
есть ли боковые ответвления;
есть ли скрытые элементы;
можно ли вернуться;
не слишком ли много тупиков;
понятно ли, где растёт риск.
```

Только после этого создаются полноценные карточки мест и переходов.

## Следующий шаг

После утверждения принципа зонирования нужно заполнить первую версию матрицы зон Layer 1 и на её основе начать проектировать места `L001–L010` и переходы `T001–T012`.
