# Flora Design Matrix Layer 1–3 (Матрица флоры Layer 1–3)

## Кратко

Flora Design Matrix Layer 1–3 — рабочая матрица флоры для первых трёх слоёв Бездны.

Документ нужен, чтобы дозаполнить флору до удобного игрового минимума и не придумывать растения хаотично.

Цель на текущем этапе:

```text
Сделать по 10 видов флоры для Layer 1, Layer 2 и Layer 3.
````

Матрица не является финальной карточкой растения. Здесь фиксируются только концепты:

- слой;
- статус;
- класс;
- игровые роли;
- угроза;
- съедобность;
- краткая идея.

Полные карточки создаются позже по шаблону флоры.

## Принцип заполнения

Растения придумываются не от эффекта, а от логики среды.

Порядок:

```
слой → биом/условие → класс растения → поведение → игровые роли → образцы
```

Образцы подробно не расписываются в матрице. Они будут прорабатываться в карточке конкретного растения.

## Классы

Используем классы из [[Flora_Classification|Классификации флоры]]:

- кормовые;
- защитные;
- хищные;
- средообразующие;
- индикаторные;
- симбиотические;
- лекарственные;
- культурные.

## Игровые роли

Основные роли:

- еда;
- ориентир;
- опасность;
- фауна-связь;
- медицина;
- материал;
- маршрутный фактор;
- лорный маркер.

Роль “образец” отдельно не используется, потому что почти любое растение может дать образец.

# Layer 1

## Профиль слоя

Первый слой — верхняя, относительно понятная и учебная среда.

Флора здесь должна быть ближе к обычной природе, но уже показывать правила Бездны: не хватать всё руками, аккуратно собирать образцы, следить за запахом, фауной, влажностью и краями маршрута.

Ожидаемый тон:

```
почти знакомо, но уже не безопасно
```

Большинство растений имеют угрозу `none` или `low`. Опасные виды встречаются, но скорее учат осторожности, чем сразу убивают.

## Заметки по Layer 1

Layer 1 должен дать игрокам базовые типы взаимодействия:

- съесть можно, но лучше понимать что именно;
- образец можно испортить грубым сбором;
- запах может привлечь фауну;
- растение может быть ориентиром;
- защитная флора учит работать в перчатках;
- маршрутная флора влияет на подъём и спуск, но не заменяет снаряжение.

# Layer 2

## Профиль слоя

Второй слой — лес, туман, вертикальность и потеря ориентации.

Флора здесь должна сильнее влиять на маршрут, навигацию и взаимодействие с фауной. Съедобные растения остаются, но чаще требуют правильного сбора или обработки.

Ожидаемый тон:

```
лес живёт по своим правилам, и человек в нём не главный
```

Здесь больше индикаторных, симбиотических и защитных растений. Угроза чаще `low` и `moderate`.

## Заметки по Layer 2

Layer 2 должен усилить значение наблюдения.

Здесь растения чаще говорят игрокам:

- где вода;
- где туман;
- где старые полости;
- где ходит фауна;
- где можно подняться;
- где опасно задерживаться.

Еда всё ещё есть, но она менее прямолинейна: требуется вымачивание, сушка, очистка или знание нужной части растения.

# Layer 3

## Профиль слоя

Третий слой — разлом, скалы, ветер, высота и редкая цепкая жизнь.

Флора здесь должна ощущаться менее обильной, но более приспособленной к экстремальным условиям. Многие растения связаны с трещинами, ветром, сухостью, цепкими корнями и аварийным выживанием.

Ожидаемый тон:

```
жизнь не украшает разлом, а цепляется за него
```

Угроза чаще связана не с ядом, а с маршрутом, высотой, нагрузкой, водой, ветром и истощением.

## Заметки по Layer 3

Layer 3 должен играть не количеством зелени, а ценностью каждого растения.

Здесь флора чаще нужна, чтобы:

- найти воду;
- понять направление ветра;
- заметить скрытую щель;
- оценить прочность маршрута;
- получить аварийную пищу;
- понять, где кто-то падал;
- не перегрузить путь.

Съедобные растения есть, но почти всегда требуют обработки или аккуратности.

## По общему ощущению

Layer 1:

```
учебный, зелёный, сравнительно понятный, с мягкими ошибками.
```

Layer 2:

```
лесной, влажный, навигационный, связанный с фауной и туманом.
```

Layer 3:

```
сухой, скальный, ветреный, цепкий, аварийный.
```

|   |   |
|---|---|
|Слой|Опасные или условно опасные|
|Layer 1|Stingnettle, Sweetcap Sprout, Hookgrass|

Эскалация выглядит логично: чем глубже, тем опасность чаще связана не с “ядом”, а с маршрутом, фауной, высотой и ошибкой подготовки.

# Следующие шаги

1. Проверить, какие концепты выглядят лишними или дублирующими.
2. Уточнить пищевые растения Layer 2.
3. Добавить 1 мягкое лекарственное растение на Layer 1 или встроить лечебный образец в уже выбранный вид.
4. После утверждения матрицы начать карточки homebrew-флоры.
5. Первым слоем создать карточки Layer 1, потому что он нужен для стартовых экспедиций.
