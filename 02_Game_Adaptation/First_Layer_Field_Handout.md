# First Layer Field Handout (Памятка по первому слою)

## Кратко

First Layer Field Handout — краткая полевая памятка для кандидатов из [[Belchero_Orphanage|приюта Белчеро]], отправляющихся на проверочный учебный маршрут в [[Layer_1|первый слой]].

В рабочем русском переводе можно использовать вариант Памятка по первому слою.

Документ предназначен для игроков как внутриигровой handout. Он не раскрывает все сведения о Бездне и не заменяет полноценный справочник Delver. Это короткая памятка перед первым спуском.

## Общие правила первого слоя

- Не отходить от учебного маршрута без причины.
- Не преследовать фауну.
- Не есть неизвестные растения.
- Не трогать яркие, липкие, колючие или пахнущие образцы голыми руками.
- Не шуметь у гнёзд, костей, следов крови и обрывков снаряжения.
- Любую странную находку сначала описать, потом брать.
- Всё, что похоже на реликвию, сдать после возвращения.
- При сомнении отметить место на карте и уходить.

## Фауна первого слоя

## Короткие заметки по существам

### Hammerbeak

Hammerbeak может казаться обычной крупной птицей, пока не подойдёшь слишком близко.

Опасность:

- клюв;
- атака сверху;
- защита гнезда;
- внезапный налёт.

Правило:

```
Увидел крупную птицу — проверь, не стоишь ли ты рядом с её гнездом.
```

### Crimson Splitjaw

Crimson Splitjaw — не учебная встреча.

Если вы видите его следы, свежие кости или слышите тяжёлое движение рядом, задача не “победить”, а уйти.

Опасность:

- прямое нападение;
- засада;
- погоня;
- паника группы.

Правило:

```
Если Crimson Splitjaw заметил тебя раньше, чем ты его, ты уже опоздал с храбростью.
```

### Silkfang

Silkfang опасен не только сам по себе, но и тем, что может превратить местность в ловушку.

Опасность:

- нити;
- узкие проходы;
- плохой обзор;
- запутывание;
- потеря времени.

Правило:

```
Нить в Бездне — это не украшение. Сначала подумай, кто её оставил.
```

### Demonfish

Demonfish связан с водой и влажными участками.

Опасность:

- мутная вода;
- внезапный рывок;
- травма руки или ноги;
- потеря образца или снаряжения.

Правило:

```
Если не видишь дна, не проверяй его пальцами.
```

### Sakawatari

Sakawatari — угроза, которую лучше заметить заранее.

Опасность:

- атака сверху;
- открытые площадки;
- перенос добычи;
- паника на узком маршруте.

Правило:

```
На открытых местах сначала смотри вверх, потом вперёд.
```

## Флора первого слоя

## Короткие заметки по флоре

### Eternal Fortunes

Eternal Fortunes выглядят красиво, но не всякая красота в Бездне предназначена для рук Delver.

Опасность не обязательно в яде.

Иногда опасность в том, что вы отвлеклись, наклонились, подошли к краю или забыли смотреть вокруг.

Правило:

```
Красивое растение сначала отмечают на карте. Потом уже решают, нужно ли к нему подходить.
```

### Sainona

Sainona можно встретить на первом слое.

Для учебного маршрута это может быть допустимый растительный образец, если наставник разрешил сбор.

Правило:

```
Если растение входит в задание — бери аккуратно.Если не входит — сначала опиши.
```

### Gintoko

Gintoko лучше считать объектом для наблюдения, а не для еды или экспериментов.

Правило:

```
Не пробовать неизвестное растение на вкус.Даже если старшие дети спорят, что “так проверяют настоящие Delvers”.
```

### Ouba

Ouba может быть полезным ориентиром или образцом, но не поводом сходить с маршрута без причины.

Правило:

```
Если растение помогает отметить маршрут — зарисуй его.Если растение заставляет тебя уйти с маршрута — подумай дважды.
```

## Что можно брать как учебный образец

Допустимые образцы:

- сухой лист;
- маленький минеральный фрагмент;
- кусочек мха;
- пустая оболочка насекомого;
- отпавший фрагмент растения;
- необычный камешек без видимого эффекта;
- предмет, который можно взять без риска.

Не брать без разрешения:

- яйца;
- живых существ;
- части свежей туши;
- яркие ягоды;
- липкие растения;
- неизвестные грибы;
- реликвии, которые двигаются, греются, светятся или звучат.

## Признаки опасности

Сразу остановиться и осмотреться, если заметили:

- свежие кости;
- обрывки ткани;
- следы крови;
- резкий запах;
- внезапную тишину;
- нити между камнями;
- много мелких следов в одном месте;
- повреждённую маршрутную отметку;
- следы когтей на камне;
- блестящий предмет рядом с гнездом или норой.
