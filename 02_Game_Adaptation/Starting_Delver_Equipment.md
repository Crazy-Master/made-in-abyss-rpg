# Starting Delver Equipment (Стартовое снаряжение искателя)

## Кратко

Starting Delver Equipment — рабочий файл о базовом снаряжении начинающего [[Delvers|искателя]] перед первым учебным спуском.

В рабочем русском переводе можно использовать вариант Стартовое снаряжение искателя.

Это уже переходный файл между каноном и будущей RPG. Он опирается на канонический образ Delvers, [[Whistles|свистков]], [[Mail_Balloons|почтовых шаров]] и игровую структуру `Made in Abyss: Binary Star Falling into Darkness`, но не задаёт точных правил, веса и цен.

Для первой мини-арки этот файл нужен, чтобы подготовка к спуску стала частью истории, а не сухим списком покупок.

| Категория    | Примеры                                            |     |
| ------------ | -------------------------------------------------- | --- |
| Навигация    | карта, маршрутный лист, заметки                    |     |
| Связь        | [[Mail_Balloons\|почтовый шар]]                    |     |
| Передвижение | верёвка, крюк, обувь, перчатки                     |     |
| Свет         | фонарь, факел, запас огня                          |     |
| Выживание    | еда, вода, бинты                                   |     |
| Добыча       | мешок для находок, простые инструменты             |     |
| Отчётность   | бланк задания, отметки, образцы                    |     |

В игре Supply Shop продаёт множество предметов для спуска: верёвки, бинты, почтовые шары, инструменты, одежду и расходники.

В каноне Delvers явно используют практическое снаряжение и работают в рамках ранговой системы, но точный стартовый набор Red Whistle не описан как единая таблица.

## Степень достоверности

Подтверждено или указано в основных справочных сводках:

- Delvers используют снаряжение для спусков;
- ранг [[Whistles|свистка]] ограничивает допустимую глубину и задачи;
- [[Mail_Balloons|почтовые шары]] используются для отправки сообщений на поверхность;
- начинающие Delvers связаны с обучением, наставниками и отчётностью;
- [[Red_Whistle|Красные свистки]] ограничены ранними зонами первого слоя.

Подтверждено или указано в игровых материалах:

- в игре есть Supply Shop;
- магазин продаёт верёвки, бинты, почтовые шары, одежду и расходники;
- часть снаряжения открывается по рангу;
- Mail Balloon имеет игровой вес 80 г и цену 500 Orth;
- Rope, Bandage и другие предметы имеют игровые цены и веса;
- игровые показатели можно использовать как черновик, но не как финальный баланс.

Требует проверки по первоисточнику:

- точный обязательный набор Red Whistle в манге;
- выдаётся ли снаряжение приютом или покупается самостоятельно;
- какие предметы считаются обязательными по правилам Гильдии;
- какие вещи дети реально несут на первых спусках;
- насколько игровые цены подходят будущей настольной версии;
- нужно ли учитывать вес предметов уже в первой мини-арке;
- где проходит граница между атмосферной подготовкой и перегрузкой игроков менеджментом.

## Базовый набор

Для первого игрового спуска можно использовать мягкий стартовый набор.

Это не финальная механика, а список предметов, которые логично иметь у Red Whistle.

| Предмет | Роль в сцене |
|---|---|
| Свисток | статус и допуск |
| Малый рюкзак | перенос находок и еды |
| Верёвка | спуск, подъём, страховка |
| Фонарь или факел | свет в тенях и проходах |
| Бинты | первая помощь |
| Вода | выживание и паузы |
| Простая еда | усталость и привалы |
| Нож или малый инструмент | бытовые задачи |
| Мешочек для реликвий | сбор находок |
| Маршрутный лист | связь с заданием |
| Почтовый шар | отчёт или экстренное сообщение |

Для первой мини-арки лучше не давать игрокам слишком большой список. Достаточно 6–8 важных предметов, остальные можно считать частью стандартного набора.

## Предметы выбора

Чтобы подготовка была интересной, можно дать каждому игроку один выбор.

Примеры:

| Выбор | Игровой смысл |
|---|---|
| Дополнительная верёвка | безопаснее маршрут, но больше вес |
| Ещё один бинт | лучше подготовка к травме |
| Запас еды | меньше риск усталости |
| Малый фонарь | удобнее в тёмных местах |
| Пустой контейнер | можно взять образец |
| Старый справочник | помощь с реликвиями, но неполные данные |
| Личный талисман | эмоциональная связь с поверхностью |

Это создаёт чувство подготовки без сложной экономики.

## Ограничения Red Whistle

Начинающий Delver не должен иметь доступ ко всему.

Ограничения могут быть социальными:

- нельзя брать дорогое снаряжение без разрешения;
- нельзя покупать опасные инструменты;
- нельзя идти глубже разрешённой зоны;
- нужно вернуть выданное имущество;
- нужно сдать найденные реликвии;
- нужно отправить отчёт или вернуться до срока.

Такие ограничения лучше работают как часть мира, а не как запрет мастера.

## Роль снаряжения в первом спуске

Снаряжение должно создавать решения, а не бухгалтерию.

Хороший подход:

- верёвка нужна в одной явной сцене;
- бинт нужен, если кто-то получил травму;
- почтовый шар нужен для отчёта;
- еда нужна для привала;
- мешок для находок делает реликвии физическими;
- свет создаёт атмосферу в тёмном проходе.

Плохой подход:

- считать каждый грамм с первой игры;
- заставлять игроков покупать 20 предметов;
- наказывать за отсутствие вещи, о которой никто не мог догадаться;
- делать подготовку длиннее самого спуска.

## Связь с приютом и магазинами

Стартовое снаряжение может прийти из трёх источников:

| Источник                              | Как ощущается                                 |
| ------------------------------------- | --------------------------------------------- |
| [[Belchero_Orphanage\|Приют Белчеро]] | учебная выдача, контроль, обязанность вернуть |
| [[Orth_Supply_Shops\|Магазины Орта]]  | выбор, цена, слухи, самостоятельность         |
| Личные вещи                           | эмоция, индивидуальность, риск потери         |

Лучше смешать все три.

Например: приют выдаёт обязательный минимум, магазин позволяет докупить одну вещь, а каждый персонаж берёт личный предмет.

## Роль в мире

Starting Delver Equipment выполняет несколько функций:

- делает спуск материальным и конкретным;
- показывает подготовку как часть культуры Delvers;
- связывает приют, Гильдию и магазины;
- создаёт первые решения игроков;
- подчёркивает, что Бездна опасна даже на первом слое;
- помогает не превращать первый спуск в абстрактное “идём в данжон”.

## Краткие игровые заметки

- Для первой игры нужен короткий список, а не полноценный инвентарь.
- У каждого предмета должна быть потенциальная сцена.
- Вес можно упоминать качественно: легко, нормально, тяжело.
- Почтовый шар лучше сделать обязательным элементом первого задания.
- Игроки должны почувствовать, что готовятся к настоящей экспедиции.
- Снаряжение должно помогать, но не решать всё автоматически.

## Возможные игровые системы

- стандартный набор;
- один личный предмет;
- один предмет выбора;
- ограничение по весу;
- выдача от приюта;
- покупка в магазине;
- потеря снаряжения;
- повреждение предмета;
- отчёт через почтовый шар;
- сдача реликвий после возвращения.
