# Charcoal Sand (Угольный песок)

## Кратко

Charcoal Sand — реликвия-материал, связанная с происхождением [[Scaled_Umbrella|Чешуйчатого зонтика]] Рико.

В рабочем русском переводе можно использовать вариант Угольный песок.

Согласно игровым и справочным материалам, Charcoal Sand был спрессован в пластины и сшит в форму зонта. После такой обработки итоговый предмет потерял класс и стал processed Relic.

Для проекта Charcoal Sand важен как пример реликвии-сырья: не готового устройства, а вещества, которое можно переработать в снаряжение.

## Степень достоверности

- Charcoal Sand описывается как реликвия третьего класса;
- Scaled Umbrella сделан из Charcoal Sand;

- пластины были сшиты в форму зонта;
- обработка снизила класс итогового предмета;

Требует проверки по первоисточнику:

## Внешний вид

Точный внешний вид Charcoal Sand до обработки требует проверки.

По названию и функции можно осторожно предположить, что это зернистый или порошкообразный материал, напоминающий угольный песок. Но это рабочая интерпретация, а не подтверждённый факт.

После обработки материал становится частью [[Scaled_Umbrella|Чешуйчатого зонтика]]:

- спрессованные пластины;
- сшитая конструкция;
- защитная форма;
- потеря исходного класса при переработке.

## Эффект

Самостоятельный эффект Charcoal Sand пока неясен.

Известное применение связано со [[Scaled_Umbrella|Scaled Umbrella]], который служит защитным предметом.

Рабочая цепочка:

1. Charcoal Sand существует как реликтовый материал.
2. Его прессуют в пластины или бруски.
3. Пластины соединяют в форму зонта.
4. Получается [[Scaled_Umbrella|Scaled Umbrella]].
5. Из-за обработки предмет теряет класс и становится processed Relic.

Важно: пока не стоит приписывать Charcoal Sand конкретную защиту, прочность или особую магическую функцию отдельно от Scaled Umbrella.

## Использование

Charcoal Sand показывает, что реликвии могут быть не только готовыми устройствами.

Некоторые реликвии могут использоваться как сырьё:

- для изготовления снаряжения;
- для ремонта;
- для ремесленной обработки;
- для продажи;
- для экспериментов;
- для создания processed Relics.

Это полезная деталь для будущей игровой адаптации, но пока не нужно превращать её в полноценную систему крафта.

## Статус реликвии

Для нашего проекта Charcoal Sand лучше считать смешанным объектом.

Причина:

- [[Scaled_Umbrella|Scaled Umbrella]] связан с основным каноном;
- подробности о материале сильнее опираются на игровые и справочные материалы;
- Charcoal Sand как отдельный объект требует проверки по манге.

Рабочее правило:

Charcoal Sand можно использовать как осторожно принятую каноническую деталь, но не строить на нём важные выводы о реликвиях без подтверждения первоисточником.

## Связь со Scaled Umbrella

[[Scaled_Umbrella|Scaled Umbrella]] — главный известный предмет, связанный с Charcoal Sand.

Эта связь важна для понимания processed Relics.

Scaled Umbrella — не просто найденная реликвия в исходной форме, а результат переработки. Это показывает, что жители Орта и Delvers могут менять найденные артефакты, приспосабливая их под практические задачи.

## Роль в мире

Charcoal Sand выполняет несколько функций:

- объясняет происхождение [[Scaled_Umbrella|Чешуйчатого зонтика]];
- показывает реликвии как сырьё;
- вводит идею processed Relics;
- связывает реликвии с ремеслом и экипировкой;
- помогает будущей системе крафта, но пока без механики;
- показывает, что обработка может снижать класс предмета.

## Краткие игровые заметки

- Charcoal Sand можно использовать как пример реликтового материала.
- Не стоит пока выдавать точные свойства без проверки.
- В будущем материал может лечь в основу простого крафта защитного снаряжения.
- Для ранней игры лучше упоминать его через [[Scaled_Umbrella|Scaled Umbrella]], а не давать игрокам напрямую.
- Важнее показать идею processed Relic, чем вводить отдельные правила.
